# CSS ANIMATIONS

**[Animate.css]**(https://animate.style/)

## Installation

```bash
npm install @shubham_17/cssanimations
```

## Usage

```reactjs
import React from "react";
import Animate from "@shubham_17/cssanimations";
class App extends React.Component {
    render() {
        return(
            <div>
                <h1>App</h1>
                <Animate animation="fadeIn">
                    <h1>Hello</h1>
                </Animate>
            </div>
        )
    }
}
```
